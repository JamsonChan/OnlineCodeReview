作業下載:

作業說明:

1. 請設計一個函式 Homework_2，它的參數有三個：matrix、y、x。
2. matrix 的型態為 list[list[int]]; y、x 的型態為 int。
3. 請依照參數將 matrix 轉換成 y * x 的二維陣列。
4. 如果可以成功轉換為新的二維陣列，回傳新的二維陣列和 True，無法則回傳原本的 matrix 和 False。
5. import 函式庫來撰寫作業，作業以0分計算。(ex. import numpy ...)
6. 最後答案 return 的型態為 list[list[list[int]], bool]。

注意事項:

  🟠 1 <= matrix[i][j]、y、x <= 10000
  🟠 1 <= len(matrix)、len(matrix[0]) <= 1000
  🟠 請參考以下範例。

範例一:

    Input : matrix = [[3,4],[5,6]], y = 1, x = 4
    Output: [[[3, 4, 5, 6]], True]
