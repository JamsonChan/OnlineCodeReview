Homework Rules

上傳作業規則說明:

1. 請同學在上傳作業時，確認是否有將 test.py 改成 s + 自己的學號.py。
--> (ex. test.py -> s1104813.py)
2. 請同學在規定的函式 and 類別中，撰寫本次作業程式碼。
--> 請避免 (ex. 更動函式參數、區域外變數)
3. 同學在繳交作業前，可以先利用Auto.bat做測試。
--> 請避免 (ex. 無限迴圈、error)
4. 請同學在繳交作業前，確認是否有將答案 return 出來。
--> (ex. return ans)
5. 系統上傳檔案的格式，只接受 .py 檔。
--> (ex. s1104813.py)
6. Time Limit Exeeded 代表程式碼有問題需要優化。
--> (作業成績:0)
7. 每次作業的上傳次數，限制為20次。
8. 請同學切勿擅自更改其任何函式 and 類別名稱。
