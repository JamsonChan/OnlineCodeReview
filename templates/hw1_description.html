Homework 1(作業說明)

⭐ 請勿擅自更改類別和方法名稱

作業下載:

作業說明:

1. Class Homework_1 它的參數有四個：list1、list2、parameter1、parameter2。
2. 請設計三個類別方法：division_list1、multiplication_list2、remainder_list1_and_list2。
3. 類別方法執行順序：(division -> multiplication -> remainder)。
4. list1、list2 的型態為 list[int]; parameter1、parameter2 的型態為 int。
5. division_list1()：list1[i] 整除 parameter1 並更新 list1。
6. multiplication_list2()：list2[i] 乘以 parameter2 並更新 list2。
7. remainder_list1_and_list2()：list1[i] 除以 list2[i] 的餘數加入新的陣列並回傳答案。
8. 最後答案 return 的型態為 list。

注意事項:

  🟠 list1、list2 中的數字可能會重複。
  🟠 list1、list2、parameter1、parameter2 的數字範圍為 0 ~ 10000。
  🟠 1 <= len(list1) == len(list2) <= 100。
  🟠 請參考以下範例。

範例一:

    Input : list1 = [3,2], list2 = [2,3], parameter1 = 2, parameter2 = 3
    Output: [1,1]

範例一解說(根據作業規定之類別方法執行順序):

  1.division_list1(): list1[0] 整除 parameter1 = 1，list1[1] 整除 parameter1 = 1，list1 = [1,1]。
  2.multiplication_list2(): list2[0] 乘以 parameter2 = 6，list2[1] 乘以 parameter2 = 9，list2 = [6, 9]。
  3.remainder_list1_and_list2(): list1[0] 除以 list2[0] 餘數 = 1，list1[1] 除以 list2[1] 餘數 = 1，return [1,1]。
 

範例二:

    Input : list1 = [1,2,3,4,5], list2 = [1,2,3,4,5], parameter1 = 2, parameter2 = 5
    Output: [0,1,1,2,2]

範例三:

    Input : list1 = [2,953,110,4], list2 = [501,1502,120,50], parameter1 = 1, parameter2 = 4
    Output: [2,953,110,4]
